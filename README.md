# DevNest

DevNest is a simple login and registration web application built using Node.js, Express, and MongoDB. The project demonstrates basic user authentication with password hashing and session-based login.

## Technologies Used
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- bcrypt
- express-session
- HTML
- CSS
- JavaScript

## Features
- User registration
- User login
- Password hashing using bcrypt
- Session-based authentication
- Logout functionality
- Simple dashboard showing the logged-in user

## Running the Project

Start the backend server:

cd server  
npm start  

The backend runs on:  
http://localhost:5050

Start the frontend:

cd client  
npx http-server -p 3000  

Open the application in your browser:  
http://localhost:3000
